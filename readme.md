## The coffee colors scheme for vim

## Install using Pathogen:

	cd ~/.vim/bundle
	git clone https://github.com/duythinht/vim-coffee

## Install using Vundle

	Plugin 'duythinht/vim-coffee'

## Modify .vimrc

	syntax enable
	colorscheme coffee
